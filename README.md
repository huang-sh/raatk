# RAATK
A python-based reduce amino acid toolkit of machine learning for protein-dependent inference.

Installation
------------
```
$ pip install raatk==1.0.0
```

or

``` 
$ pip install git+https://github.com/huang-sh/raatk.git@master -U
```
